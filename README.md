1. **Client-facing presence**

2. **Demonstration of work**

   - Demonstrates my approach to structure, clarity, and usability  
   - Acts as a living example of my web development skills  

## Scope

- Simple layout
- Fast load times
- Minimal dependencies

This reflects the kind of low-cost, efficient, and maintainable websites I aim to offer to clients.
